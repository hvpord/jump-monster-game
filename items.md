# 跳怪兽道具数据手册（v1.9.11）

> 时间单位默认按 **60fps** 换算：60 帧 = 1 秒。

## 通用规则

- **技能书（📖）** 是永久叠加强化，吃到后一直累加，**死亡后清零**。
- 普通道具在顶部状态栏显示剩余时间 / 次数。
- 锤子、盾牌、足球、无人机统一在消失前 **2 秒** 开始硬闪烁（完全出现 / 完全消失，每 10 帧切换一次），随后消失。
- 乘车（吉普车）或带球跑（足球未踢出）时，吃到 **翅膀无效**。

## 技能书

| 项目 | 数值 |
|------|------|
| 生成方式 | 独立计时器，每 **60–100 秒** 自动生成一本 |
| 外形 | 紫色技能书 |
| 作用 | 每吃一本，`player.skillBooks + 1`，为多个道具提供加成 |
| 显示 | 顶部按数量重复显示 📖 |

### 技能书对各道具的加成一览

| 道具 | 加成方式 |
|------|----------|
| 锤子 | 持续时间 +1 秒/本；可击飞次数 +1/本 |
| 翅膀 | 持续时间 +1 秒/本 |
| 盾牌 | 持续时间 +1 秒/本 |
| 吉普车 | 初始弹药 +1 发/本；再次吃到车时补充弹药 +1/本 |
| 足球 | 带球持续时间 +1 秒/本 |
| 火箭 | 同时发射数量 +1/本 |
| 无人机 | 最大数量每 3 本 +1；攻击间隔按乘法递减（见无人机章节） |
| 车载护盾 | 最大层数每 5 本 +1 |

## 普通道具生成

| 项目 | 数值 |
|------|------|
| 密度倍率 | `1.03^(分数/10)`（每 10 分在当前密度基础上 +3%） |
| 生成间隔倍率 | `1 / 密度倍率` |
| 最小怪物间距 | 基础 200，随密度增加按 `200 / sqrt(密度倍率)` 动态缩小，**暂不设下限** |
| 普通道具生成间隔 | 基础 5–12 秒（300–720 帧） |
| 分数影响 | 每 100 分缩短 1 秒（60 帧），最短不低于 2–4 秒（120–240 帧） |
| 随机池 | 8 种道具等概率（各 12.5%） |
| 同屏上限 | 最多 3 个普通道具 |

道具池：`hammer`、`wings`、`stone`、`shield`、`jeep`、`football`、`rocket`、`uav`

## 道具优先级 / 受击顺序

当角色持有多个效果时，受到怪物碰撞的结算顺序：

1. **锤子**（无敌击飞怪物）
2. **吉普车车载护盾**（消耗一层）
3. **吉普车本体碰撞箱**（车状态直接撞飞地面怪物）
4. **普通盾牌**（抵挡一次）
5. **死亡**（无任何保护时）
